# Product Identity

EFOOTBALL COIN MANAGER PRO

This is NOT a coin tracker.

This is NOT a gaming companion.

This is NOT a pack simulator.

This is a Behavioral Decision Support System.

The purpose of the application is to:

* Reduce impulsive spins.
* Increase 900 coin goal completion.
* Reduce regret.
* Improve discipline.
* Improve long-term decision making.

All implementation decisions must support these objectives.

# Architecture Rules

Follow the architecture from the audit report.

Separate:

* UI
* Business Logic
* Data Layer

No business logic inside visual components.
