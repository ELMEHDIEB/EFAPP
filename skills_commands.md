# 🛠️ Hub des Compétences (Skills)

> [!TIP]
> **Comment ça marche ?** Tapez simplement la commande souhaitée dans le chat pour déclencher le workflow correspondant, ou décrivez votre besoin et je l'activerai pour vous !

## 🏗️ Conception & Architecture

| Commande | Rôle de l'IA | Description de l'action |
|:---|:---|:---|
| 🎯 `/office-hours` | **CEO / Produit** | Affiner une idée, brainstormer et définir le vrai besoin. |
| 🔎 `/plan-ceo-review` | **CEO** | Challenger la portée et la vision d'une fonctionnalité. |
| 🏗️ `/plan-eng-review` | **Tech Lead** | Valider l'architecture, la sécurité et les flux de données. |
| 🎨 `/plan-design-review` | **Designer UI/UX** | Revoir le plan sous l'angle de l'expérience utilisateur. |
| 🤖 `/autoplan` | **Pipeline** | Lancer toutes les révisions de plan automatiquement. |
| 📝 `/spec` | **Product Manager** | Rédiger une spécification technique pour le backlog. |

---

## 🎨 Design & UI/UX

| Commande | Rôle de l'IA | Description de l'action |
|:---|:---|:---|
| 🖌️ `/design-consultation` | **Design Partner** | Créer un design system ou une identité visuelle complète. |
| ✨ `/design-review` | **UI Engineer** | Améliorer les détails visuels (espacements, couleurs). |
| 🧩 `shadcn init` | **Intégrateur UI** | Initialiser un projet avec `shadcn/ui`. |
| ➕ `add [composant]` | **Intégrateur UI** | Ajouter un composant UI spécifique. |
| 🎨 *(Via le chat)* | **UI/UX Pro Max** | Demander 50 styles, palettes de couleurs premium, ou polices. |

## 🐞 Test, Qualité & Débogage

| Commande | Rôle de l'IA | Description de l'action |
|:---|:---|:---|
| 🕵️ `/investigate` | **Debugger** | Trouver la cause racine d'un bug complexe et le corriger. |
| 🧪 `/qa` | **Testeur QA** | Tester le comportement du site web et fixer les bugs. |
| 👁️ `/qa-only` | **Testeur QA** | Signaler les bugs sans modifier le code source. |
| 🔍 `/review` | **Staff Engineer** | Faire une revue de code approfondie sur vos changements. |
| 🛡️ `/cso` | **Responsable Sécurité**| Lancer un audit de sécurité poussé (normes OWASP). |
