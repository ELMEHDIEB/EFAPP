# EFAPP

## Overview

EFAPP (eFootball Coin Manager) is a desktop application designed to help users manage and track multiple eFootball accounts and coin balances.

## Technology Stack

* React
* Vite
* SQLite
* JavaScript
* Tailwind CSS

## Core Features

* Multi-account management
* Coin balance tracking
* Spin tracking
* Analytics dashboard
* Mental coaching module
* Post-loss recovery module
* PIN protection
* Local data storage

## Development Rules

* Maintain clean architecture.
* Keep all data local unless explicitly specified.
* Prioritize performance and usability.
* Preserve backward compatibility with existing data.
* Document major changes in CHANGELOG.md.

## Current Goal

Transform EFAPP into a professional-grade eFootball management platform with advanced analytics and account monitoring.
